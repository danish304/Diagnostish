# Diagnostish

Портативная консольная утилита на C# для автоматизации первичной диагностики и сбора технических характеристик компьютера с флеш-накопителя.

🚀 Основной функционал

1. Сбор аппаратных данных: Выгрузка информации о процессоре (модель, ядра), оперативной памяти (суммарный объем в ГБ, скорсоть), видеокартах и накопителях через Windows Management Instrumentation (WMI).
2. Автоматизация прав доступа: Встроенный манифест приложения приложения (UAC) автоматически запрашивает права администратора для корректного чтения низкоуровневых датчиков.
3. Портативность (Self-Contained): Программа собирается без внешних зависимостей и работает на любом ПК с Windows без предварительно установленного .NET.
4. Удобство: Результаты выводятся в кастомный цветовой интерфейс консоли.

🏗️ Архитектура и технологии

1. Проект написан на языке C# (.NET 10).
2. Models/ — независимые сущности для хранения и транспортировки данных (HWReport).
3. Services/ — изолированные сервисы для логики сканирования (HWCheck), работы с дисками и запуска внешних процессов.
4. System.Management — NuGet-пакет для безопасного общения с Win32-классами операционной системы.

📋 Как использовать

1. Скачайте последний релиз.
2. Поместите скомпилированный файл .exe на сервисную флешку.
3. Запустите утилиту на целевом ПК — она сама запросит права админа, соберет базовые ТТХ и откроет сопутствующие портативные программы.
