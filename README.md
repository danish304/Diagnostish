# Diagnostish

Портативная консольная утилита на C# для автоматизации первичной диагностики и сбора технических характеристик компьютера. Разработана специально для запуска с сервисных флеш-накопителей.

🚀 Основной функционал

Сбор аппаратных данных: Низкоуровневый опрос компонентов (процессор, оперативная память, видеокарты, накопители) через Windows Management Instrumentation (WMI).
Автоматизация прав доступа: Встроенный манифест приложения (UAC) автоматически запрашивает права администратора для корректного чтения системных датчиков.
Автономность (Self-Contained): Утилита собирается в один исполняемый файл со всеми зависимостями. Работает на любом ПК с Windows без предварительной установки .NET.
Информативность: Результаты сканирования структурированы и выводятся в кастомный цветовой интерфейс консоли.

🏗️ Архитектура и технологии

Технологический стек: Проект написан на языке C# (.NET 10).
Models/ — независимые сущности для хранения и транспортировки данных (HWReport, OSReport).
Services/ — изолированные сервисы для бизнес-логики сканирования компонентов (HWCheck, OSCheck) и обработки исключений.
Views/ — слой представления данных (IPrintHW, IPrintOS, PrintToConsole), построенный на интерфейсах для гибкой подмены формата вывода (консоль, файл, GUI).
Controllers/ — управляющий слой (Program), координирующий жизненный цикл приложения и потоки данных.
System.Management — NuGet-пакет для безопасного взаимодействия с Win32-классами операционной системы.

📦 Как использовать

Скачайте последний релиз утилиты во вкладке Releases.
Поместите скомпилированный файл .exe на сервисную флешку.
Запустите утилиту на целевом ПК — она автоматически запросит права администратора, соберет базовые ТТХ и подготовит систему к дальнейшей диагностике.
