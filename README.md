# Diagnostish

Портативная консольная утилита на C# для автоматизации первичной диагностики и сбора технических характеристик (ТТХ) компьютера с флеш-накопителя мастера.

🚀 Основной функционал

Сбор аппаратных данных: Выгрузка информации о процессоре (модель, ядра), оперативной памяти (суммарный объем в ГБ), видеокартах и накопителях через Windows Management Instrumentation (WMI).
Автоматизация прав доступа: Встроенный манифест приложения приложения (UAC) автоматически запрашивает права администратора для корректного чтения низкоуровневых датчиков.
Портативность (Self-Contained): Программа собирается без внешних зависимостей и работает на любом ПК с Windows без предварительно установленного .NET.
---------SOON-----------Запуск внешнего софта: Модуль автоматического вызова тяжелого портативного диагностического софта (CrystalDiskMark, CPU-Z, GPU-Z) по относительным путям.
Удобство: Результаты выводятся в кастомный цветовой интерфейс консоли.

🏗️ Архитектура и технологии

Проект написан на языке C# (.NET 10) с упором на чистую архитектуру и принципы ООП (SOLID/SRP):

Models/ — независимые сущности для хранения и транспортировки данных (HWReport).
Services/ — изолированные сервисы для логики сканирования (HWCheck), работы с дисками и запуска внешних процессов.
System.Management — NuGet-пакет для безопасного общения с Win32-классами операционной системы.

📋 Как использовать

Скомпилируйте проект в режиме Release как Self-contained.
Поместите скомпилированный файл .exe на сервисную флешку.
Запустите утилиту на целевом ПК — она сама запросит права админа, соберет базовые ТТХ и откроет сопутствующие портативные программы.
